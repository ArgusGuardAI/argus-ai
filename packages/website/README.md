# WhaleShield Website

Landing page for WhaleShield - The First Whale Token With REAL Utility.

## Preview Locally

```bash
# Option 1: Using npx serve
npx serve .

# Option 2: Open directly in browser
open index.html
```

## Deploy to Cloudflare Pages

1. Go to [Cloudflare Pages](https://pages.cloudflare.com/)
2. Connect your GitHub repo
3. Set build settings:
   - Build command: (leave empty)
   - Build output directory: `packages/website`
4. Deploy!

## Deploy to Vercel

1. Go to [Vercel](https://vercel.com/)
2. Import your GitHub repo
3. Set root directory to `packages/website`
4. Deploy!

## Deploy to Netlify

1. Go to [Netlify](https://netlify.com/)
2. Drag and drop the `packages/website` folder
3. Done!

## Customization

- Edit `index.html` to update content
- Uses Tailwind CSS via CDN (no build step needed)
- Update social links to your actual Twitter/Discord/Telegram
- Replace placeholder stats with real numbers
- Add actual extension download link when ready

## Files

- `index.html` - Main landing page
- `package.json` - Package info
- `README.md` - This file
