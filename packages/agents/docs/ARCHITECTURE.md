## System Overview

The Argus Agent System is a **multi-agent AI architecture** designed for real-time Solana token analysis. The system emphasizes:

- **Autonomy**: Agents operate independently with their own reasoning loops
- **Collaboration**: Agents communicate via message passing to share intelligence
- **Efficiency**: CPU-only inference with extreme data compression
- **Learning**: Continuous improvement through outcome tracking

### Cosine Similarity Search

```typescript
// How similarity search works
function cosineSimilarity(a: Float32Array, b: Float32Array): number {
  let dotProduct = 0;
  let normA = 0;
  let normB = 0;

  for (let i = 0; i < a.length; i++) {
    dotProduct += a[i] * b[i];
    normA += a[i] * a[i];
    normB += b[i] * b[i];
  }

  return dotProduct / (Math.sqrt(normA) * Math.sqrt(normB));
}
```
